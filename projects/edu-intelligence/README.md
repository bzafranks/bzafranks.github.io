# Bahamas Educational Intelligence Platform
### A Data-Driven Instruction & School Improvement System

Project #6 in the **Curriculum Innovation & Instructional Design Portfolio** of Dr. Barbara Z.
Franks — and the first **Instructional Leadership** project. Built on the portfolio's shared
design system; deployed at **https://bzafranks.github.io/projects/edu-intelligence/**.

## 📘 Project Overview
A commercial-grade educational intelligence platform that turns education data into actionable
insight from the **classroom to the national level**. Unlike the other portfolio projects, it is
**not curriculum- or lesson-centric** — it demonstrates learning analytics, dashboard design, and
AI decision support for instructional leadership and school improvement.

## 🎯 Educational Challenge
Schools collect abundant data but underuse it. Teachers lack timely insight, leaders can't see
trends, and the Ministry needs meaningful information to allocate resources. This platform
transforms data into intelligence that improves decisions at every level.

## 📊 Learning Analytics Philosophy
The right insight, to the right person, at the right time — **timely, actionable, and
explainable**. Every insight pairs with a recommended action; AI shows its reasoning and a human
decides.

## ✨ Platform Features
- **6 role-based dashboards:** Teacher, Principal, District, **Ministry (executive)**, Student, Parent
- **Learning analytics** — mastery heat maps, growth/longitudinal charts, cohort filters, an
  **early-warning system**, and **attendance & engagement** analytics
- **AI Recommendation Engine** — an interactive, **explainable** insight engine + an
  **AI executive-briefing generator** (teacher → ministry)
- **Curriculum monitoring** (coverage, pacing, fidelity) and **assessment intelligence**
  (item analysis, misconceptions, reliability)
- **School improvement planning** (goals, action plans, progress monitoring)
- **Platform architecture** + **data governance** (privacy, roles, ethical/explainable AI)
- **Implementation guide**, consulting **case study**, and **reflection**
- Animated KPIs, heat maps, trendlines, segmented controls, live indicators, and the shared
  **accessibility toolbar**

## 🛠️ Technology Stack
Shared portfolio **design-system.css** + **portfolio.js** (with the reusable data-viz components
and accessibility toolbar). A project script **`eip.js`** powers animated KPI count-ups, the AI
executive-briefing generator, the explainable AI insight engine, and dashboard filters/segmented
controls. Vanilla HTML/CSS/JS, inline SVG charts, `localStorage`, GitHub Pages.
**All data is illustrative sample data; AI is simulated client-side for demonstration.**

## ▶ Run Locally
Open this folder's `index.html`, or serve the repo root (`python -m http.server`).

## 🚀 Deployment
