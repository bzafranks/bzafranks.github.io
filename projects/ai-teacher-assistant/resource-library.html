🗂️ Resource Library

Resource Library

Everything you save from any AI tool is stored right here — in your browser. Generate a lesson, assessment, rubric, message, or feedback note with any tool, press Save to Library, and it lands here. Search and filter to find exactly what you need.

How it works

💾

Saved as you work

Any output you generate in the Lesson Planner, Assessment Generator, Rubric Builder, Communication, or Feedback tools can be saved with one click. Each resource keeps its title, type, and full text so you can reuse or adapt it later.

🔍

Search and filter

Type in the search box to match on titles and content, or use the type filters above to narrow to just lessons, assessments, rubrics, communications, or feedback. Results update instantly as you type.

🔒

Stored in your browser

This demonstration keeps your library entirely client-side in localStorage — nothing is uploaded to a server. Clearing your browser data or using a different device starts with a fresh, empty library. You can also clear it anytime from Settings.

☁️

In a production version

A live release would sync your library to a secure teacher account and integrate with your LMS — so resources follow you across devices, can be shared with a team or PLC, and can be pushed directly into Google Classroom, Canvas, or Microsoft Teams. Soon

👤 Human-in-the-loop Saved items are AI-assisted drafts. Review each one for accuracy, bias, and fit before sharing it with students or families.
