Personalized Learning Paths

Five Paths. One Standard. Every Learner.

Every lesson generates multiple pathways. Students don't get a watered-down or inflated curriculum — they reach the same standards by the route that fits them now, and move between paths as mastery grows.

The Pathways

A–E, Built Into Every Lesson

Path A Targeted Support

Needs additional support

Re-teaching video, heavily scaffolded practice with hints, vocabulary pre-teach, and sentence frames. Builds the foundation before independence.

Path B On-Grade

On-grade level

Core instruction, targeted practice, and peer discussion — the on-track route with timely feedback.

Path C Advanced

Advanced learner

Compressed instruction plus deeper application tasks and a harder text to stretch mastery.

Path D Independent

Independent challenge

Self-directed inquiry with a complex problem, lighter scaffolding, and a mastery checkpoint.

Path E Creative

Creative extension

Demonstrate the skill in a new medium — podcast, comic, video, or design — for transfer and engagement.

Movement is the point

Paths aren't tracks. A student on Path A who masters a skill moves to B or C the same day. A student who stumbles on C drops back for a quick re-teach, then returns. The AI suggests; the student and teacher decide.

The logic (simplified)

if score < 60:   → Path A  (re-teach + scaffold)
elif score < 80: → Path B  (targeted practice)
elif score < 90: → Path C  (application)
else:            → Path D/E (challenge + create)
# Always overridable by the teacher.
