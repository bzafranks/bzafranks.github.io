Project Overview

A Curriculum That Survives Staffing Disruptions

When a teacher is absent, a position goes unfilled, or learning moves online overnight, instruction usually suffers. This project demonstrates a different outcome: a curriculum engineered from the ground up so that every lesson can be delivered by any qualified adult — or by the student independently — without losing rigor or engagement.

Designed for Continuity

One Lesson, Four Ways to Deliver It

Each lesson is intentionally designed to run in any of these environments — the heart of the continuity strategy.

🏫

Face-to-Face

The full lesson supports a classroom teacher with modeling, discussion, and guided practice.

🟢

Synchronous Online

Live virtual delivery with shared video, breakout discussion, and real-time checks.

🔄

Asynchronous

Self-paced, fully self-explaining lessons with embedded video and auto-feedback.

🤝

Hybrid

Any blend of the above — a substitute facilitates while students drive the digital core.
