# Educational Leadership Consulting Playbook

**Educational Leadership & School Improvement Suite · CAPSTONE · Project #15**
A comprehensive **consulting methodology for leading sustainable educational transformation** — from initial diagnosis through long-term sustainability. The capstone of the Leadership Suite: it integrates every major framework into one coherent transformation strategy, the way world-class firms partner with ministries and school systems.

Part of the portfolio of **Dr. Barbara Z. Franks**, built on the shared portfolio design system.

🔗 **Live:** https://bzafranks.github.io/projects/leadership-consulting-playbook/

---

## Educational Transformation Philosophy

Successful school improvement happens when leadership development, strategic planning, instructional excellence, organizational culture, teacher retention, continuous improvement, and innovation are woven into one coherent strategy — diagnosed honestly, sequenced deliberately, and built to sustain.

## Consulting methodology

A six-phase lifecycle — **Discover · Diagnose · Design · Develop · Deliver · Sustain** — supported by an interactive maturity model, a Transformation Readiness diagnostic across six dimensions (Leadership, Culture, Instruction, Strategy, Talent, Innovation), and a transformation blueprint generator.

> All figures throughout the site are **illustrative sample data**. The diagnostic saves locally in the browser via `localStorage`. Client success stories are fictional/illustrative.

## Research foundation

Integrates Dr. Franks' doctoral dissertation (leadership behaviors → teacher retention → school performance) with transformational leadership, school-improvement research, strategic leadership, continuous improvement, organizational development, and future-ready leadership — translated into practical consulting tools.

## Key features

- **Transformation Readiness diagnostic** → a Readiness Index (Initiating → Transforming) with frameworks matched to priorities.
- **Interactive maturity model** — click to profile an organization across six dimensions.
- **Transformation blueprint generator** — school, district, system/ministry, turnaround, strategic-improvement, and 100-day launch plans.
- **Leadership Solutions hub** integrating all thirteen Leadership Suite frameworks (Future-Ready Schools Framework noted as planned).

## Technology stack

Vanilla HTML5 / CSS3 / JavaScript — no frameworks, no build step. Shared `../../assets/css/design-system.css` + `../../assets/js/portfolio.js`; project `style.css` (graphite-teal `#243643` + signature teal `#2f9e8f`, app-shell, dark theme) and `script.js` (readiness diagnostic, maturity model, blueprint generator, gauges, bars, count-ups, drill-downs, notifications, command palette, theme toggle). Persistence via `localStorage` (`clp:` prefix).
