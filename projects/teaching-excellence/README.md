# Teaching Excellence & Instructional Leadership Framework
### An Evidence-Based System for Developing Exceptional Teaching Through Leadership, Coaching & Continuous Improvement

The third project in the **Educational Leadership & School Improvement Suite**, part of Dr. Barbara Z.
Franks' portfolio. Built on the shared portfolio design system; deployed at
**https://bzafranks.github.io/projects/teaching-excellence/**.

## ЁЯУШ Project Overview
A comprehensive *instructional leadership system* тАФ not a teacher-evaluation model. It helps leaders
build a culture of instructional excellence through six integrated domains, an interactive teaching
model, coaching cycles, growth-focused feedback, professional learning, curriculum leadership, and
data-informed continuous improvement.

## ЁЯОп The Instructional Leadership Challenge
Teaching quality is the single greatest school-based influence on student achievement тАФ but improving
it takes more than observation. It takes a coordinated system of coaching, feedback, professional
learning, curriculum alignment, and data use.

## ЁЯзн Framework Philosophy

## ЁЯФм Research Foundation
