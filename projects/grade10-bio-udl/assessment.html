Assessment System

Many Ways to Show Mastery

UDL assessment lets every learner demonstrate the same Biology outcomes through the mode that fits them — while keeping the standard the same for all.

The Assessment Stack

Diagnostic → Formative → Practical → Performance → Summative

🩺

Diagnostic

Reveals prior knowledge to plan instruction and supports.

🔁

Formative

Knowledge checks, virtual labs, and reflections with instant feedback.

🔬

Practical Lab

Microscopy, sampling, and investigation skills (virtual or physical).

🛠️

Performance Tasks

Authentic application — field studies, models, presentations.

🏁

Summative

End-of-unit demonstration of the official outcomes.

🗂️

Portfolio

A collection of evidence across modes and units.
